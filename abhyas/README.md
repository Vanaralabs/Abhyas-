# ⚡ Abhyas

**अभ्यास** — Practice. A free, interactive coding platform. Write real code in your browser — no install, no setup.

*Built by [vanaralabs](https://github.com/vanaralabs)*

![React](https://img.shields.io/badge/React-18-61DAFB?logo=react)
![Vite](https://img.shields.io/badge/Vite-5-646CFF?logo=vite)
![TailwindCSS](https://img.shields.io/badge/Tailwind-3-06B6D4?logo=tailwindcss)

## Features

- 🎯 **12 Languages** — Python, JavaScript, HTML, CSS, TypeScript, C++, Java, SQL, R, Git, Docker, AI Prompts
- 💻 **In-Browser Code Editor** — Monaco Editor (same as VS Code)
- 📖 **80 Interactive Lessons** — Theory → Code Challenge → Quiz
- 🎮 **Code Arena** — Timed coding challenges with difficulty levels
- 🔥 **Streaks & XP** — Daily streak tracking, XP system, levels
- 🏆 **Badges** — Unlock achievements as you progress
- 💡 **Hints** — Premade hints for every challenge
- 📱 **Responsive** — Works on desktop, tablet, and mobile
- 🌙 **Futuristic UI** — Dark theme, neon accents, glassmorphism, animations

## Tech Stack

| Layer | Tech |
|-------|------|
| Framework | React 18 |
| Build | Vite 5 |
| Styling | Tailwind CSS 3 |
| Code Editor | Monaco Editor |
| Animations | Framer Motion |
| State | Zustand + localStorage |
| Icons | Lucide React |
| Deploy | GitHub Pages |

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub → GitHub Actions auto-builds and deploys to GitHub Pages.
